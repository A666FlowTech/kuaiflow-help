# ISO 8601表达式

## 一、ISO 8601持续时间格式

这种格式适合定期周周期必发起、例如每月1号1点0分0秒发起；

#### 1、格式

ISO 8601持续时间格式如下：
P(n)Y(n)M(n)DT(n)H(n)M(n)S


其中：

- P是持续时间指示符，始终放置在持续时间的开头位置

- T是表示时间元素开始的指示符

- `(n)` 为时间值

| 符号 | 含义 |
| --- | --- |
| Y | 年 |
| M | 月 |
| D | 日 |
| W | 星期 |
| H | 时 |
| M | 分 |
| S | 秒 |

表示的持续时间为：三年八个月五天，十二小时三十分钟六秒。

## 二、ISO 8601日期格式

适合定时指定一个固定日期时间发起

#### 1、格式

ISO 8601日期格式如下：
YYYY-MM-DDThh:mm:ss[.mmm]TZD


其中：

- `YYYY` 表示四位数的年份

- `MM` 表示两位数的月份

- `DD` 表示两位数的天(day of the month)，从01到31

- `T` 是用来指示时间元素的开始字符

- `hh` 表示两位数的小时，从00到23，不包括 `AM/PM`

- `mm` 表示两位数的分钟，从00到59

- `ss` 表示两位数的秒，从00到59

- `mmm` 表示三位数的毫秒数，从000到999

- `TZD` 表示时区指示符： `Z` 或 `+hh:mm` 或 `-hh:mm`， `+` 或 `-` 表示时区距离UTC(世界标准时间)时区多远。例如：

CST(中国标准时间)： `UTC +08:00`，EST(东部标准时间)： `UTC -05:00`，CST(中部标准时间)： `UTC -06:00`。

#### 2、样例
2012-03-29T10:05:45-06:00


表示：中部标准时间2012年3月29日10:05:45。
